Versione accessibile di LODE - Modalità Off-Line

Questa finestra fornisce le informazioni necessarie per utilizzare LODE off-line. Prima di procedere assicurarsi di comprendere le informaizoni presenti in questa pagina poiché sarà necessario riavviare il browser internet.

Per avviare la lezione LODE in maniera accessibile seguire i passi seguenti:

  • Leggi le istruzioni che seguono quindi chiudi TUTTE le finestre del browser prima di procedere.
  • Muoversi nella cartella dove si è scaricata la lezione.
  • Entrare nella cartella ' contents '
  • Entrare nella cartella ' accessibility '
  • Se utilizzi Windows a questo punto seleziona ' WinLoad '
  • Se utilizzi Mac OS o Linux avvia il file setup.jar'
NOTA: Per utilizzare l'installatore guidato è necessario avere installato JAVA sul vostro sistema. Se l'applicativo non si avvia seguendo la procedura descritta sopra verificare di avere installato JAVA quindi riprovare.
